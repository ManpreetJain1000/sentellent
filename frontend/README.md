# Frontend

React + TypeScript + Tailwind UI for Sentellent Phase 1.

## Scope

- Landing page
- Google OAuth sign-in and callback handling
- Agentic chat interface with memory sidebar
- Workspace connection status

## Run locally

```bash
npm install
cp .env.example .env
npm run dev
```

## Tests

```bash
npm test
npm run build
```

## Environment

| Variable | Purpose |
| --- | --- |
| `VITE_API_BASE_URL` | Backend API base URL |
